3. Python Base64

编码：b64encode

			
import base64
base64.b64encode('This is an encoded string')

此示例将显示：

'VGhpcyBpcyBhbiBlbmNvZGVkIHN0cmluZw=='

解码：

			
import base64
base64.b64decode('VGhpcyBpcyBhbiBlbmNvZGVkIHN0cmluZw==')

此示例将显示：

This is an encoded string
