13.8. Javascript Framework

javascript是面向過程的，只要請引用.js檔案即可訪問他的方法（function），並且傳統方式會定義很多全局變數。如果大量使用javascript難免會出現變數覆蓋，或function同名。

所以我們要將javascript封裝成class，另一點我們也需要面向對象支持。下面是幾個常用的JS開發框架（javascript framework）。

  • prototype

  • jQuery

  • mootools

  • script.aculo.us

  • Dojo

  • MochiKit

  • rico

Javascript GUI

  • ExtJS

  • qooxdoo
