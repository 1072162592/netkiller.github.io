防止日誌檔案本修改，加入某些屬性。

1：chattr 命令可以使某個檔案屬性改變
     chattr  +a /var/log/array.log   只允許追加，不允許減少。
     chattr  -a /var/log/array.log   去掉屬性。
2：lsattr 命令查看chattr屬性
# lsattr /var/log/array.log
-----a------- /var/log/array.log
