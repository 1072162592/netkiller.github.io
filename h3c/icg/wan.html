4.3.1. DHCP 动态IP地址

			
<Netkiller>system-view
System View: return to User View with Ctrl+Z.

interface Ethernet0/0
 port link-mode route
 nat outbound
 ip address dhcp-alloc
 ipsec policy navigator
#
