1. php 文件

1.1. 格式

使用 UNIX 风格换行, 请在你的编辑器内调整

UNIX (LF或"\n")
MAC OS (CR 或"\r")
Windows CRLF \r\n

1.2. 编码

UTF-8

有些IDE环境 UTF-8 BOM

1.3. 取出行尾的空格以及多余的换行符

一个空格占用一个字节,换行符Window是两个字节\r\n, Unix与Mac占用一个字节

1.4. php 标签

禁止这样使用

			
<?
...
?>

正确的使用方法

			
<?php
...

or

<?php
...
?>
