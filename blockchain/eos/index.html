26.1. EOS

https://eos.io

EOS是一种全新的区块链架构，旨在实现分布式应用的性能拓展。EOS 项目的目标是实现一个类似操作系统一样的支撑应用程序的区块链架构。该架构可以提供账户，身份认证，数据库，异步通信以及可在数以百计的 CPU 或群集上的程序调度。该技术的最终形式是一个区块链体系架构，该区块链每秒可以支持数百万个交易，同时普通用户无需支付使用费用。

26.1. EOS

		
cd /usr/local/src/
git clone https://github.com/EOSIO/eos --recursive
cd eos/
git submodule update --init --recursive

./eosio_build.sh
