2.1.2. example

例 57.2. server.py

$ cat server.py
import zmq  
context = zmq.Context()  
socket = context.socket(zmq.REP)  
socket.bind("tcp://127.0.0.1:5000")  

while True:  
    msg = socket.recv()  
    print "Got", msg  
    socket.send(msg)

例 57.3. client.py

$ cat client.py 
import zmq  
context = zmq.Context()  
socket = context.socket(zmq.REQ)  
socket.connect("tcp://127.0.0.1:5000")

for i in range(10):  
    msg = "msg %s" % i  
    socket.send(msg)  
    print "Sending", msg
