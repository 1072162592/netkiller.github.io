3. Language

默认语言

export LANG=en_US
export LC_ALL=en_US

永久更改

sudo vi /etc/default/locale

LANG="en_US.UTF-8"
LANGUAGE="en_US:en"

改为中文环境

sudo apt-get install language-support-zh
LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh"
