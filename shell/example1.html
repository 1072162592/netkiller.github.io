6. Shell 技巧

6.1. 行转列，再批评

echo "abc def gfh ijk"| sed "s:\ :\n:g" |grep -w gfh

6.2. for vs while

echo "aaa bbb ccc" > test.txt
echo "ddd eee fff" >> test.txt
            
for line in $(cat test.txt)
do
	echo $line
done
			
cat test.txt| while read line
do
	echo $line
done
