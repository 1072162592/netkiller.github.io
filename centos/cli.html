41.2. docker 命令

41.2.1. run

run

$ sudo docker run ubuntu:14.04 /bin/echo 'Hello world'
Hello world

41.2.2. start / stop / restart

sudo docker start silly_bohr
silly_bohr

$ sudo docker stop silly_bohr
silly_bohr

$ sudo docker restart silly_bohr
silly_bohr
