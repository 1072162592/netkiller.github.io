4.5. MMDVMCal

MMDVMCal 是用来调试 DMR 的工具

		
cd /usr/local/src 
git clone https://github.com/g4klx/MMDVMCal.git
cd MMDVMCal
make

sudo cp MMDVMCal /srv/mmdvm/bin

# sudo /srv/mmdvm/bin/MMDVMCal /dev/ttyACM0		
		
		
		
neo@netkiller /usr/local/src/MMDVMCal % ./MMDVMCal /dev/ttyUSB0
